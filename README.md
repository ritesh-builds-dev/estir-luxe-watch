## 📋 Project Overview
ESTIR WATCH is a front-end masterpiece designed to showcase luxury timepieces. The project emphasizes visual storytelling through clean layouts and interactive elements, ensuring a premium user experience across all devices.

## ✨ Key Features
- **Modern Minimalist UI:** A dark-themed, sleek aesthetic tailored for luxury branding.
- **Advanced Typography:** Strategic use of `Michroma`, `Montserrat`, and `Josefin Sans` for a professional look.
- **Pixel-Perfect Responsiveness:** Meticulously optimized for devices from large desktops down to **320px** mobile screens.
- **Dynamic Interactions:** Custom hover effects on product cards and smooth transitions.
- **Comprehensive Sections:** Includes a Hero Banner, Brand Showcase, Featured Collections, and a detailed Blog section.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for optimal SEO and accessibility.
- **CSS3:** Custom properties (Variables), Flexbox, Grid, and complex Media Queries.
- **Iconography:** Remix Icons integration for a sharp, modern interface.

## 📸 Design Highlights
- **Mobile-First Approach:** Ensuring the design remains "Anti-Fragile" and consistent on any viewport.
- **Clean Documentation:** Code written with readability and maintenance in mind.
